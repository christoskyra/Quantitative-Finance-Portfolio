# Quantitative Finance Portfolio

A collection of Python scripts demonstrating core concepts in Financial Risk Management and Quantitative Analysis.

## Contents
1. **Volatility Analysis:** Calculation of historical volatility and distribution analysis.
2. **Monte Carlo Simulation:** Simulating future stock price paths using Geometric Brownian Motion (GBM) to estimate Value at Risk (VaR).
3. **CAPM Analysis:** Implementing the Capital Asset Pricing Model using Linear Regression (`scipy.stats`) to calculate Beta and Alpha.

## Tech Stack
- Python (Pandas, NumPy, Matplotlib, SciPy, yfinance)
